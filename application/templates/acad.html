Friday, 25th August, 2023

Lesson: Organizing Drawings

Layers: When drawings are organized into layers, the following object properties can be managed easily: Visibility in drawing environment, colors, line weights Print visibility and so on. Model and Layouts: It is usually convenient to prepare title panels in layout views and the rest of the drawing in the model view. Automatic Organizations: When highlighting lots of objects in the drawing area, the autocad software automatically arranges them in object groups such as polylines, text, mtext, aligned dimensions, rotated dimensions, etc.
